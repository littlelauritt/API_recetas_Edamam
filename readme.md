# APP con Edamam

_Ejercicio Api recetas con Edamam._


## Comenzando 🚀

_Aplicacion que se conecta a la API de Edamam y devuelve recetas. El límite de busquedas es de 100 y solo se pueden hacer 10 busquedas por minuto._


## Despliegue 📦

_npm run build_


## Construido con 🛠️
